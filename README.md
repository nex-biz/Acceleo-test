# Acceleo-test
Test Java code generation from UML metamodel


### Prérequis
Installer Eclipse (distribution "modeling")
Installer le plugin "Acceleo"


### Procédure pour l'exécution

Pour exécuter la transformation test, il faut :
1. faire un clic droit sur le projet 
2. sélectionner "Run As"
3. sélectionner "Launch Acceleo Application"

Dans la nouvelle fenêtre qui s'ouvre, il faut renseigner les champs suivants :
1. Project : sélectionner le projet en cours
2. Main Class : sélectionner la classe "Uml2java"
3. Model : sélectionner le fichier "model.uml"
4. Target: nommer le répertoire à créer (celui contiendra les fichiers sources Java)
